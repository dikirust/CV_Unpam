## ===== TROUBLESHOOTING =====
## If dataset is not found:
## - Check datasets/train/ folder exists
## - Check subfolders for 5 classes

## If training takes too long:
## - Reduce EPOCHS value in main script
## - Increase BATCH_SIZE for faster processing

## If memory error:
## - Reduce IMG_SIZE to 48 or 32
## - Reduce BATCH_SIZE to 8
## - Reduce number of samples loaded
