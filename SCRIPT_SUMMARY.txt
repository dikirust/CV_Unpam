RINGKASAN SCRIPT WASTE CLASSIFICATION
====================================

File utama: waste_classification_main.py

SCRIPT INI BERISI:

STEP 1: MEMUAT DATA
- Load semua gambar dari folder datasets/train/
- Split data: 70% training, 15% validation, 15% test
- Preprocessing: resize ke 64x64, normalisasi pixel

STEP 2: DEFINISIKAN ARSITEKTUR
Model 1 - Custom CNN:
  - 3 Convolutional blocks (32 -> 64 -> 128 filters)
  - Dengan BatchNormalization dan Dropout
  - Parameters: ~300K
  - Cocok untuk training dari awal

Model 2 - MobileNetV2 (Lightweight):
  - Transfer learning dari ImageNet
  - Depthwise separable convolutions
  - Parameters: lebih kecil, lebih efisien
  - Cocok untuk device dengan resource terbatas

STEP 3: MELATIH KEDUA MODEL
- Custom CNN training
- MobileNetV2 training
- Menggunakan EarlyStopping dan ReduceLROnPlateau
- Menyimpan training history

STEP 4: VISUALISASI TRAINING CURVES
- Plot accuracy dan loss untuk Custom CNN
- Plot accuracy dan loss untuk MobileNetV2
- Perbandingan kedua model

OUTPUT:
- 01_custom_cnn_training.png - Training curves Custom CNN
- 02_mobilenetv2_training.png - Training curves MobileNetV2
- 03_training_comparison.png - Perbandingan training curves

STEP 5: EVALUASI DI TEST SET
- Hitung accuracy, precision, recall, F1-score
- Confusion matrix untuk setiap model
- Per-class accuracy
- Classification report

STEP 6: VISUALISASI HASIL EVALUASI
- 04_cm_custom_cnn.png - Confusion matrix Custom CNN
- 05_cm_mobilenetv2.png - Confusion matrix MobileNetV2
- 06_metrics_comparison.png - Perbandingan metrik (Accuracy, Precision, Recall, F1)
- 07_per_class_accuracy.png - Per-class accuracy comparison

STEP 7: RINGKASAN PERBANDINGAN
- 08_summary_comparison.png - Summary dengan 4 metrik utama:
  * Accuracy comparison
  * Model parameters comparison
  * Training time comparison
  * F1-Score comparison

OUTPUT FILES:
1. JSON files di folder output/:
   - split_info.json (informasi dataset split)
   - models_info.json (informasi arsitektur model)
   - training_info.json (hasil training)
   - evaluation_results.json (hasil evaluasi)

2. PNG files di folder output/report/:
   - 01-08 visualisasi

3. Model files di folder output/models/:
   - custom_cnn.h5
   - mobilenetv2.h5

4. Pickle files di folder output/:
   - history_custom_cnn.pkl
   - history_mobilenetv2.pkl

CARA MENJALANKAN:
python waste_classification_main.py
